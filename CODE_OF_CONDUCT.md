# Code of Conduct

## Our Pledge

We are committed to making GreenPass an inclusive, welcoming, and harassment-free project for everyone, regardless of experience, background, identity, or perspective.

## Enforcement

Violations may result in:
1. Warning
2. Temporary ban
3. Permanent ban

**Report issues** to project maintainers via GitHub Issues (mark as private) or email.

## Scope

This Code of Conduct applies to all project spaces including GitHub issues, discussions, pull requests, and any public representation of the project.
